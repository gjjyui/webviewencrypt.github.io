Add a Navigation Group

Use the Navigation Group List page to add a new Navigation Group as follows:

  1. Click Navigation Groups in the System Tree in the left pane to open the Navigation Groups List page.

  2. Type a name for the new group in the Navigation Group Name input box.
  3. Click Upload to browse for a background graphic file or click Gallery to select an image from any previously uploaded images.

  4. Click Add Group to create the new Navigation Group.

    • The new Navigation Group name appears under Navigation Groups in the System Tree.

    • The new Navigation Group with its background image opens on the Group.

You can now design your Navigation Group by adding objects and links.
