Add objects, keywords, and animations to a Navigation Group

Add objects, keywords, and animations to a Navigation Group using its Group page.

To access the Group page, click a Navigation Group in the System Tree to open it. Click the Group button if the Group page is not visible.

You can add items to a Navigation Group as follows:

  • To add objects, select Object in the Add drop-down menu, or click the magnifying glass icon next to a device in the System Tree.
  • To add keywords, select Keyword in the Add drop-down menu.
  • To add animations, select Animation in the Add drop-down menu.

Add objects to a Navigation Group

Use the following steps to add objects to a Navigation Group:

  1. Click the Navigation Group name in the System Tree that you want to modify to open the Group page for the Navigation Group. Click the Group button if the Group page is not visible.

  2. Use one of the following procedures:

    1. Click Add and then click Object to open the Devices, Objects, and Search Results panes.

      — or —

    2. Click the magnifying glass icon beside a device in the System Tree to open the Devices, Objects, and Search Results panes.

    When you use the Add menu, the Devices, Objects, and Search Results panes open for all devices in the system. When you use the magnifying glass method, the same panes open but the entry you clicked in the System Tree is selected in the Devices pane.

  3. Select one or more check boxes in the Devices and Objects panes to filter the entries in the Search Results pane.

  4. Drag one or more objects from the Search Results pane to the Navigation Group canvas.

    Use Ctrl + click or Shift + click to select more than one object to drag to the Navigation Group.

The items are automatically saved. You can reposition them anywhere on the canvas.

Add keywords to a Navigation Group

Use the following steps to add keywords to the Navigation Group:

  1. Click Add and then click Keyword to open the Keyword pane.

  2. Click a keyword in the Keyword pane and drag it to the Navigation Group canvas.

    Use Ctrl + click or Shift + click to select more than one keyword object to drag to the Navigation Group canvas.

The items are automatically saved. You can reposition them anywhere on the canvas.

Add animations to a Navigation Group

Use the following steps to add animations to the Navigation Group:

  1. To add a new animation:

    1. Select Upload RCA From Workstation and browse to the location of the RCA file.

      By default, the files in the folder location are filtered to show only RCA files. The uploading of JSON files is not supported.

    2. Click an RCA file name, and click Open to add the file path to the input box and to display the animation in the preview area.

    3. Drag the animation from the preview area to the Navigation Group canvas.

    When you drag and drop an animation on the Navigation Group, it retains its default size.

  2. To add a previously uploaded animation:

    1. Select Select Previously Uploaded RCA and click the down arrow to display the list of animations.

      The animation names are listed in alphabetical order.

      You can also type a partial name in the Select input box to filter the list of previously uploaded animations by name.

    2. Click an animation in the list to display it in the preview area, and drag it to the Navigation Group canvas.

      Note: If you reload an animation that has been uploaded previously, it will replace that previously uploaded animation on the server. Ensure that you back up the previously uploaded animation if you changed it, otherwise your changes will be overwritten.

The items are automatically saved. You can reposition them anywhere on the canvas.
