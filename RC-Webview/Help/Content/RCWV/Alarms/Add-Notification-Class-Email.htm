Add a Notification Class email

Add a notification class alarm email using the Notification Class Emails worksheet.

Note: Only controllers configured using the RC-Toolkit MSet Tool with Simple Mail Transfer Protocol (SMTP) Server credentials can send emails.

Follow this procedure to add a notification class email:

  1. In the System Tree, expand a system node and click a controller.
  2. In the flyout menu, click Notification Classes to see the Notification Classes worksheet.
  3. In the Notification Classes worksheet, select a Notification Class and click in the Emails column to open the Notification Class Emails worksheet for that Notification Class.
  4. On the worksheet, click the Add icon ,or click in an empty row to create a new email destination.
  5. Enter the appropriate information in the worksheet.
  6. Click OK when complete.
