Notification Classes worksheet

Use the Notification Classes worksheet to configure the alarm priorities, email notifications, and destinations for every BACnet device that initiates alarms.

Note: Only controllers configured using the RC-Toolkit MSet tool with Simple Mail Transfer Protocol (SMTP) Server credentials can send notification emails.

To display the Notification Classes worksheet, click a BACnet-compatible controller and then click Notification Classes.

The Notification Classes worksheet contains the following columns:

Name

The name of the Notification Class object. Click to modify.

Alarm Type

The type number assigned to the notification class. Alarm type is used to distinguish both the severity and email destinations of various alarms.

Priority

Opens the Notification Class Priorities dialog box, used to define alarm priorities according to their event type. Priority determines the relative importance of each alarm for that Alarm Type. The lower the priority number, the more critical the nature of the alarm. For example, alarms that require immediate action should be given a priority of 1. There are 255 priority levels available, with the first six configured by default.

Note: By default, alarm priorities are set to the default priority for that alarm type as defined in the Alarm Configuration dialog box in RC-Studio.

Using the Notification Class Priorities dialog box, you can associate alarm events with priorities as follows:

  • To Alarm — The BACnet alarm is activated and the event is distributed to the configured destination for the notification class.
  • To Fault — The BACnet alarm is activated for an object entering a fault state and the event is distributed to the configured destination for the notification class. The objects are generally integrated peripherals, sensors, or networked I/O.
  • To Normal — The BACnet alarm is reset and the event is distributed to the configured destination for the notification class.

Click Ack Required to require the user who acknowledges the alarm to enter a reason.

Description

Type a detailed description of the Notification Class.

Destinations

Opens the Notification Class Destination dialog box, which lists destinations in a worksheet.

Emails

Opens the Notification Class Email dialog box, which lists email recipients for email notifications.
