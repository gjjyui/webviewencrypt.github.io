Request an After Hours Event

An After Hours Event is the same as a Special Event, except an After Hours Event must be requested by an operator and approved by a user with the appropriate permissions.

Use the following steps to request a new After Hours Event:

  1. Click an Enterprise Schedule in the Navigation Tree to open the Enterprise Schedule viewer.
  2. Select the Schedule from the Schedule View pane on the right.
  3. Click Summary to change the Schedule to Edit mode. An After Hours Event can now be added in one of two ways:
    • Click and drag the cursor on the Schedule to select a time increment. Enter the Event details in the Edit Event Details pane.
    • Click the Edit Event Details tab in the top right of the window, then enter the Event details as follows:

      Event Type — Choose Special Event or Default Event. Special Events have more configurable options. Default Events are weekly Events.

      Title — Enter a title for the Event.

      Repeat — Click the drop-down list box to select how often the Event will repeat.

      From/To — Enter a start and end date for the Event.

      Start/End — Enter a start and end time for the Event.

      Priority — Normal or High priority.

      Value — Select the value based on the type of Schedule.

      Reason — Enter a reason for the Event.

      Applies to — This provides a link to the Schedules that are affected by this Event.

      Exclude From Report — When this is selected, the Event is not included in an After Hours Report.

      Note: Be aware of the hourly rate and total cost of the Event. If the administrator has set a cost for the Event, you will be billed this amount upon Event approval.

  1. When you are finished, click the Save icon to apply the changes.
