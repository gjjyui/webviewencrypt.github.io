Provide Enterprise Schedule permissions to users

Modify Enterprise Schedule permissions for a single user or multiple users using the User List page. With the appropriate permissions, users can create and modify Schedules and After Hours events.

Access the User List page in one of two ways:

  • Click the Users node in the System Tree.
  • Click a user name in the System Tree, then click the User List button.

Single users

You may wish to update the Enterprise Schedules permissions for a single user for a variety of reasons. For example, that user may get a promotion which allows access to new parts of the facility, or perhaps it simply makes sense to change the amount of access to existing Schedules for a specific user.

Use the following steps to provide a single user access to an Enterprise Schedule:

  1. Click the user name in the User List to go to their Login Info page.
  2. In the bottom right of the user's Login Info page, click the All Schedules link or View Schedules link next to Permitted Schedules. The Enterprise Schedule Permissions page appears.
  3. Select an Enterprise Schedule from the drop-down list box.
  4. Select or clear the check box for the corresponding Schedule object. To select all Schedule objects, click the Select All icon in the top right corner of the window. Selecting a parent Schedule object automatically selects the child Schedule objects as well. Click the Save icon in the top right corner of the window to save your selections.

Multiple users

You may wish to update multiple users at once if, for example, a new weightlifting room is added to a gymnasium and all users should be provided permission to turn on the heat and light in the room. Rather than having to add the Schedule for the new room to each user individually, you can add the permission to multiple users at once.

Use the following steps to update multiple users:

  1. Select check boxes of multiple users from the User List.
  2. Click the Enterprise Schedule Permissions icon in the top left of the window to perform a batch update to Enterprise Schedules permissions. Select an Enterprise Schedule from the drop-down list.
  3. Select or clear the check box for the corresponding Schedule. To select all Schedules, click the Select All icon in the top right corner of the window. Selecting a parent Schedule automatically selects the child Schedules as well. Click the Save icon in the top right corner of the window to save.

Use the following steps to update multiple users with a user role:

  1. While viewing the User List, click Role List to open the Role List page.
  2. Select the check box of the user roles that will inherit the Enterprise Schedule.
  3. Click the Enterprise Schedule Permissions icon in the top left of the window to update the Enterprise Schedules permissions. Select an Enterprise Schedule from the drop-down list.
  4. Select or clear the check box for the corresponding Schedule. To select all Schedules, click the Select All icon in the top right corner of the window. Selecting a parent Schedule automatically selects the child Schedules as well. Click the Save icon in the top right corner of the window to save.
