Enterprise Schedule Tree page

Administrators can modify Schedule objects within Enterprise Schedules from the Schedule Tree.

The Schedule Tree is an easy-to-use interface for organizing and modifying the Schedule objects contained within an Enterprise Schedule. To access the Schedule Tree for an Enterprise Schedule, click the Enterprise Schedule name in the System Tree, and then click Schedule Tree on the Settings page.

Schedules can be indented under other Schedules to create a parent-child relationship. Special Events added to a parent Schedule are inherited by the child Schedules.

Adding or modifying an Event affects the current Schedule object and any child Schedule objects below it in the Enterprise Schedule hierarchy. Changes to a lower-level Schedule object are not applied to a higher-level Schedule object.

The Enterprise Schedule Tree page provides the following functions:

Icons

Click the View icon to open the Enterprise Schedule viewer.

Click the Select All icon to select all Schedule objects in the Schedule Tree.

Click the Delete icon to remove any selected objects from the Schedule Tree.

Click the Save icon to save any changes to the Schedule Tree.

Click the Cancel icon to cancel any changes to the Schedule Tree.

Click the Edit icon on the Schedule object in the Schedule Tree to display the options for modifying the attributes of a Schedule object. The following options are available:
 

Edit Schedule Name

Click to modify the Schedule object name. You can also click the Schedule object name to modify the Schedule name.

 

Default Color

Click to return to the default color of the Schedule object. This only affects the color if you have previously changed the color.

 

Set Schedule Color

Click to open the color picker and select a color for the Schedule object. You can also click the color square preceding the Schedule object name to open the color picker and change the color.
 

Set Rate

Click Set Rate to set an hourly rate for Special Events associated with this Schedule object. If you set the rate of a child Schedule object, its parent Schedule object will have this rate added to it's total.

 

Delete

Delete the Schedule object from the Enterprise Schedule.
