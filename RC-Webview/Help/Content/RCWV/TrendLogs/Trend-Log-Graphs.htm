Trend Log viewer

View Singlepoint and Multi-Point Trend Logs in the Trend Log viewer using either of the following methods:

  • Click a link on a System Group.
  • Click a Trend Log in the Navigation Tree.

Figure: Single-point Trend Logs

 

Update Mode — Click to the Trend Log page in update mode and retrieves the latest Trend data from the controller every five seconds.

Show Data/Show Graph — Display the Trend Log data in either graph mode (displayed above) or data mode. Data mode displays current Trend Log data in a listed text format. Click to switch between views.

Previous/Next — Click to move the time range shown on the x-axis to the left or right by one entire increment of the x-axis, allowing you to view past Trend Log information, or return to current Trend Log information.

Legend — Click to display a sub-menu of the Trend Log points. Select or clear the check boxes to add or remove values from the current display. Select Cross-hairs to change the appearance of the pointer.

Data graph line — Displays the object data in graph form

X-axis Cross-hairs intersection data — Displays the X-axis data for wherever the cross-hairs are positioned.

Zoom Bar/Handles — Represents the range of data displayed in the Trend Log. Click the Zoom Bar and drag it to the left or right to adjust the time range forward or backward in history or click and drag the handles to zoom in and decrease the time range.

Unit Scale Control — Change a Trend Log Y-axis range for each unit according to the minimum and maximum values contained in the data for that unit. For example, the maximum range that is displayed for degrees Celsius may be increased or decreased to make the view more readable, while other units may be left at their default display range.

Y-axis Cross-hairs intersection data — Displays the Y-axis data for wherever the Cross-hairs are positioned.

Reliable Controls Protocol (RCP) Trend Log

For RCP Trend Logs, use these controls to modify the Log view as follows:

Update — Places the Trend Log page in update mode and retrieves the latest Trend data from the controller every five seconds.

Show Data/Show Graph — Displays current Trend Log data in text format. Use the toggle button to switch to the graphical view.
