Batch updating multiple users

Apply a single change to multiple user accounts quickly using a batch update operation. For example, you can assign a Navigation Tree, set a Home Group, or adjust Restrict to Systems properties for multiple users by selecting them on the User List page and applying a batch update. Only one property change can be made for each batch update operation.

Note: Use caution when performing a batch update operation. When you select one or more users on the User List page and then select an update option, RC-WebView updates the batch immediately.

Use the following steps to batch update multiple user simultaneously:

  1. On the System Tree click Users to see the User List.
  2. Click the Filter by Systems drop-down arrow to choose a user from a specific system, or perform a Search based on user name, Display Name, Navigation Tree, After Hours Report Template, Home Group, or Role. On the bottom task bar select the amount of user names to be displayed per page whether 10, 25, 50 or 100 (default: 25).
  3. Select the check boxes for the user names on the list that are to be included in the update. Click Select All to select all users.
  4. On the menu bar use the drop-down arrow to select the property to change for all selected users. Available options are:
    • Assign Navigation Tree
    • After Hours Template
    • Set Home Group
    • Restrict to System(s).

    You can also click Enterprise Schedule Permissions to choose Enterprise Schedules that the selected users will be able to access.

  5. Click Save to apply changes.
