Create Events in an Enterprise Schedule

Events implement a default behavior that repeats for each day of the week. An Event created in an Enterprise Schedule is written to the BACnet Schedule on the related controller. If a Special Event exists for the same time, then the Special Event takes precedence.

Complete the following steps to create a new Event in a Schedule object in an Enterprise Schedule:

  1. On the System Tree, click an Enterprise Schedule to open its Settings page, click Schedule Tree, and then click View to open the Enterprise Schedule viewer page.
  2. In the Schedule View pane, select the Schedule name to display its Calendar. If you select a parent Schedule, any new Event will automatically be applied to its child Schedules.
  3. Click Summary to put the Schedule in Edit mode. Add a default Event in one of two ways:

    • Click and drag the cursor on the Schedule to select a time increment. Use the left and right arrows to change the dates, and scroll up and down to view all hours. Enter the Event details in the Edit Event Details pane, and choose Default Event in the Event Type drop-down box.

    • Click the Edit Event Details in the pane on the right, then enter the Event details as follows:

      Event Type — Choose Default Event.

      Title — Enter a title for the Event.

      Days of the week — Indicates the day on which the default Event occurs.

      Start/End — Enter a start and end time for the Event.

      Value — Select the value based on the type of Schedule.

      Applies to — This provides a link to the Schedules that are affected by this Event.

      Exclude From Report — When this is selected, the Event is not included in an After Hours Report.

  1. To save your entry click the Save icon at the top right or click Create below Exclude From Report in the Edit Event Details pane.
