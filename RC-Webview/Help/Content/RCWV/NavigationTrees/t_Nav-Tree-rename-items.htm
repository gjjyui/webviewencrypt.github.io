Rename items in a Navigation Tree

Rename items from a Navigation Tree in the Navigation Tree Settings page. To access the Navigation Tree Settings page, click the node of the Navigation Tree in the System Tree.

Use the following steps to rename items in a Navigation Tree:

  1. In the Navigation Tree Settings page, click the name of the item you wish to rename. The name field becomes editable.
  2. Enter the new name and press Enter to apply.
  3. Click Save to save your changes, or click Cancel to undo the changes.
