Menu bar options

The menu bar is located at the top right of the banner, and is accessible throughout the application. The contents of the Options menu varies based on the information currently displayed in the main workspace.

The contents of the Options menu, their descriptions, and where they are available in the RC-WebView interface are described as follows:

Change Password

Click to change the password for the current user.

Available from all locations

Change Approval Password

Click to change the approval password, which is required to validate a change to connected system.

Print

Opens the operating system print dialog to print trend log charts or data.

Available from:

  • System Groups
  • Navigation Groups
  • Enterprise Schedules
  • BACnet Schedules
  • Weekly Schedules
  • Audit Trail
  • Trend Logs
  • Runtime Logs
  • After Hours Report
  • After Hours Reports Summary
  • Alarms
  • Alarms Summary

Audit Trail

Opens the Audit Trail worksheet. If you are managing multiple Enterprise Websites, select an Enterprise Website in the menu before clicking Audit Trail.

Available from:

  • Options > Enterprise Website name > Audit Trail
  • Navigation Tree > Any object in the Navigation Tree > Options > Audit Trail

Alarms

Opens the Alarms worksheets. If you are managing multiple Enterprise Websites, select an Enterprise Website in the menu before clicking Alarms.

Available from:

  • Options > Enterprise Website name > Alarms
  • Navigation Tree > Any object in the Navigation Tree > Options > Alarms

Alarm History

Opens the Alarm History worksheet. If you are managing multiple Enterprise Websites, select an Enterprise Website in the menu before clicking Alarm History.

Available from:

  • Options > Enterprise Website name > Alarm History
  • Navigation Tree > Any object in the Navigation Tree > Options > Alarm History

Language

Changes the language displayed by the user interface. Supports: English, German, Spanish, French, Portuguese, Chinese (Simplified), Chinese (Traditional), and Browser Default.

Available from all locations.

Theme

Changes the visual theme of the user interface. Available themes: Gray, Black, and White.

Available when you access an operator website in RC-WebView, then click Options.

Font Size

Adjusts the font size. Choices include Normal and Large.

Available when you access an operator website in RC-WebView, then click Options.

Set System Time

Sets the System Time (date and time of the connected system).

Only available from pages connected to the system.

Hide/Show Toolbar

Removes or displays the toolbar.

Available from:

  • Home Group
  • System Group
  • Enterprise Schedule
  • Enterprise Schedules
  • After Hours Reports
  • After Hours Report Summaries
  • After Hours Requests
  • Points Report Summaries
  • Points Reports
  • Trend Logs
  • Runtime Reports

Hide/Show Navigation Tree

Removes or displays the Navigation Tree.

Available from:

  • Navigation Group
  • System Group
  • Home Group
  • Enterprise Schedules
  • After Hours Reports
  • After Hours Report Summaries
  • After Hours Requests
  • Points Report Summaries
  • Points Reports
  • Trend Logs
  • Runtime Reports

Synchronize BACnet Alarms

Queries the network for current BACnet alarms.

Available from the Alarms worksheet

Unacknowledged BACnet Alarms

Checks the alarm database to see if there are any unacknowledged alarms.

Available from the Alarms worksheet

Help menu and log off

Show ToolTips

When enabled, Tooltips appear - information items (blue circles with an i) throughout the interface. Click the Tooltip icon for specific help.

User guide...

Opens the software manual in HTML.

About RC-WebView

Provides details of the current version.

Log Off

Logs off the current user and displays the RC-WebView log on screen.
