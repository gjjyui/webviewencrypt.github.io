WS-OFF

This command is not supported by the MACH-ProAir, MACH-ProZone, or any controllers running version 8 firmware.

Function

Finds when an RCP Weekly Schedule is set to turn off. Not valid for BACnet Schedules; use the the SCHED function instead.

Format

WS-OFF( exp1 , exp2 )

Where

 

 
  • exp1 is the weekly schedule row number from the Weekly Schedules worksheet
  • exp2 is the desired time off field. 1–4 are valid values.

Example

10 VAR1 = WS-OFF( 1 , 3 ) : REM Returns the third time-off field for WS1
