FLOAT

Converts an analog 0-100 percent value into floating control using two binary outputs.

Statement

Provides an analog (0-100%) to floating control (open/close/idle) actuator control function for use when actuator position feedback is not available. It features a built-in actuator position calculation based on the on time of the two outputs and the specified drive time of the actuator. When actuator position is commanded to 100% or 0% during normal operation and also during a reset operation, the FLOAT function overdrives by 25 percent to ensure the accuracy of its calculation. Note: FLOAT parameters are updated only when the statement is executed, so for best operation it should be executed every scan.

Format

position = FLOAT( open-output, close-output, loop, drive-time, deadband, reset )

Where

position is the calculated actuator position.

open-output is a BO used to drive the actuator open. The function turns on the BO when required to open the actuator.

close-output is a BO used to drive the actuator closed. The function turns on the BO when required to close the actuator.

loop is an AV or loop object with a current value to be used as the required position of the actuator. The function attempts to drive the actuator to this value.

drive-time is the actuator manufacturer's specification for the time to drive the actuator from 0 to 100%, expressed in seconds from 0-300.

deadband is an allowable percentage of deviation between the loop value and the calculated position value that does not require actuator repositioning. This value represents the full deadband and is internally divided by two to check on either side of current position.

reset is an object that when set to 1 initiates a reset and recalibration routine. The variable can be set back to 0 immediately and should be kept at 0 during normal operation. Triggering reset drives the actuator fully open to 100%, with overdrive to ensure the actuator has reached a fully open position, and subsequently closes it to 0%, with overdrive to ensure actuator has reached a fully closed position.

Example

10 FLOAT-DMP-POS = FLOAT( BO1 , BO2 , LOOP1 , 120 , 10 ,
   AH1-SYNC )
