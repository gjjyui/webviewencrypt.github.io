RC‑WebView features

RC‑WebView provides the following features. For a complete and up-to-date listing of product features, refer to the catalog and submittal sheets available from the Reliable Controls Dealer Support Center.

BACnet Operator Workstation

  • Take advantage of a web server that meets or exceeds the BACnet Operator Workstation (B-OWS) profile.
  • Access any third-party internet-connected BACnet system configured with System Groups.
  • Access and change inputs, outputs, values, schedules, programs, groups, alarms, Single-point Trend Logs, Multipoint Trend Logs, and Notification Classes.
  • Create, modify, and delete Single-point Trend Logs, Multipoint Trend Logs, Notification Classes, Point Reports, and programs.
  • Graph and print trend logs and runtime reports.
  • Render HTML5 animations on System Groups.

BACnet File Services

  • When enabled in RC-Studio 3.6.1 or later, RC-WebView can store and distribute files in the Resource Files section of Resource Manager.

Enterprise Websites

  • Create multiple Enterprise Websites.
  • Add multiple existing BACnet IP systems to a single Enterprise Website. IP connections are limited by the number of licenses you purchase.
  • Use a single sign-on to access an Enterprise Website.
  • Access Active Directory authentication or use local authentication module.
  • Create scalable, fully customizable user interfaces with Enterprise Schedules, Navigation Groups, Navigation Trees, and users.

Enterprise Schedules

  • Create hierarchical and exception schedules across multiple devices and multiple systems.
  • Use block schedules to assign any existing schedule to a named list you can enable or disable as needed.
  • Automatically or manually clear expired events using the Cleanup Configuration service.
  • View a list of pending updates using the Enterprise Schedule Queue feature.

Navigation Groups

  • Create nested background image maps, and populate them with multiple objects and links for seamless navigation.
  • Link URLs and RC-WebView visual elements such as System Groups, schedules, other Navigation Groups, and alarm summaries.
  • Navigation Groups support live data from numerous BACnet and RCP-protocol objects.
  • Use HMTL animations with live data on Navigation Groups.

Navigation Trees

  • Create a unique menu for any user or user role within the Enterprise Website.
  • Link URLs and RC-WebView visual elements such as System Groups, Schedules, other Navigation Groups, and alarm summaries.

Gallery

  • Access and manage banner images, Navigation Group graphics, and logos.

After-Hours Services

  • Building tenants can request, and managers can approve, zone operation outside regularly scheduled periods.
  • Report feature summarizes after-hours activity and associated costs.

Panel File Backup and Restore

  • Back up and restore BACnet devices.
  • Schedule automatic network backups.

Programs

  • Use the RC-WebView editor to write program code, including syntax highlighting, trace flags, and watch lists.

Title 21 CFR Part 11 Security Features

  • Change Approval option enforces users to authenticate their system changes.
  • Password Expiry option enforces new password creation at a specified interval.
  • Lockout Settings option enforces an automatic lockout of user credentials after a specified number of failed logon attempts.
  • Watermark option enforces a configurable watermark image on all pages printed from the system.
  • Security Certificate option enforces a default or configurable security certificate to validate exported logs.
  • Supports TLS 1.2.

Users

  • Flexible permissions allow administrators to configure access to objects and services by user or user role.

Alarms

  • Display global alarm summary from all systems within an Enterprise Website.
  • Allow user-configurable or context-sensitive alarm filtering to support Title 21 CFR Part 11 requirements.
  • Display numerous alarm properties by user, including time, type, message, and state.
  • Reposition alarm columns.
  • Use convenient pop-up messages for new alarms.
  • Configure alarm properties, broadcast destinations, and email recipients using the Notification Classes worksheet.
  • Configure all aspects of alarm emails, including the title and body, and include links that direct users to specific areas of RC-WebView related to the alarm.

Audit Trail

  • Automatically record every change made by users, including administrators.
  • Control access with password-enabled security levels.
  • Track alarm properties, and allow user-configurable or context-sensitive filtering to support Title 21 CFR Part 11 requirements.
  • Digitally sign audit logs to ensure authenticity to support Title 21 CFR Part 11 requirements.

Users

  • Flexible permissions allow administrators to configure access to objects and services by user or user role.

Language Support

  • Support for English, French, Spanish, Portuguese, German, and Chinese.

License and Updates Subscription

  • License allows up to 500 IP connections per server.
  • 12-month update subscription included with initial license.
  • Software continues to function regardless of subscription state.
  • No penalty to renew expired subscriptions.

Certifications

  • BTL listed (B-OWS).
