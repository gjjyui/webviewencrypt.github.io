Alarms worksheet

View all current alarms and their information using the Alarms worksheet. The worksheet columns can be adjusted for size, position, and sort order, similar to a spreadsheet, and set to display or hide parameters as required.

There are two ways to access the Alarms worksheet:

    • Click Options in the menu bar and then click System name > Alarms.
    • Click the Notifications icon then click View All Alarms.

The Alarms worksheet includes the following toolbar items:

Filter by System Click and select one or more systems in the list to show only their alarms. Systems are listed only when more than one system is configured.
Load — Retrieves all current alarms from the RC-WebView server.
Ack — Acknowledges the currently highlighted alarm.
Clear — Clears the currently highlighted alarm and removes it from the Alarms worksheet.

Export — Exports a .csv file that contains the alarm information listed in the worksheet.

This option is not available when the enforce change approval option is enabled on the Enterprise Website Security page.

Export As Excel — Exports an Excel file that contains the alarm information listed in the worksheet.

An exported Excel file is digitally signed using the signature certificate specified on the Enterprise Website Security page.
Clear All Filters — Removes any filters applied in the worksheet.
Toggle Columns — Opens the shortcut menu to choose the columns displayed in the Alarms worksheet. Select or clear check boxes to either display or hide columns.
Select All — Selects all alarms in the Alarms worksheet and action changes to Deselect All.
Deselect All - Clears the selection of all alarms in the Alarms worksheet and the action changes to Select All.

Worksheet columns

The Alarms worksheet contains the following columns:

Alarm Time

The time and date the alarm was generated at the controller level.

Alarm Type

Corresponds to the alarm types defined on the Type tab in the Alarm Configuration dialog box in RC‑Studio (Alarms > Alarm Configuration). Alarm types 0 and 1 are defined as system and general, but 62 other alarm types can also be configured.

Message

The message associated with the alarm.

State

There are four different states:

  • Alarm — Unacknowledged alarm appears in white text on a red background.
  • Ack — Acknowledged alarm appears in red text on a normal background.
  • Reset — An automatically resolved alarm appears in green text on a normal background. To clear the alarm, highlight the row and click Ack on the worksheet toolbar.
  • Clear — Alarm text appears in white text on a normal background. After an alarm is cleared, the alarm no longer appears in the Alarms worksheet. View cleared alarms in the Alarm History worksheet.

Priority

Determines the relative importance of each alarm. The lower the priority number, the more critical the nature of the alarm. For example, alarms that involve life safety may be assigned a priority number of 1. There are 255 priority levels available.

Source

Displays the number of the controller or BACnet object that generated the alarm.

System

View alarms from multiple systems.

Time Acknowledged

The time the alarm was acknowledged.

Time Reset

The time the alarm condition no longer existed.

Action

The action defined in Control-BASIC. It can be disabled, prompted, or unconditional as defined in the Alarm Configuration dialog box.
