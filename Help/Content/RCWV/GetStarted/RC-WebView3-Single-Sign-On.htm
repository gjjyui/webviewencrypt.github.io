System and Navigation Trees

RC-WebView organizes traditional administration features using a System Tree, and traditional operator features using a Navigation Tree.  You can think of the System Tree as an administrator view, and the Navigation Tree as the operator view.

The tabs at the top of the left Tree pane are used for switching between administrator and operator views:

Click the System Tree icon to access administrator features. The System Tree provides access to features that allow you to build Enterprise Schedules, Navigation Groups, Navigation Trees, and Trend Log worksheets.

Click the Navigation Tree icon to access operator features.
