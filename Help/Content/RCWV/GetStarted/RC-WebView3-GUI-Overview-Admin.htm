RC-WebView user interface

The RC-WebView user interface is divided into six main sections, where content may vary depending on whether you are logged on as an administrator or as an operator.

Figure: Sample RC-WebView navigation screen

Banner — The banner displays the name of the Enterprise Website. Any combination of text or graphics can be placed here. Clicking on the banner returns you to the Navigation Group associated with the Enterprise Website.

Tree pane— The Tree pane contains the System Tree, used by RC-WebView administrators, and the Navigation Tree, used by operators. The Navigation Tree can be customized for each group of users.

Main workspace — By default, the main workspace occupies the majority of the user interface. It displays the various services and objects within the product.

Notifications menu— Click Notifications to open the Notifications menu. This provides information on recent alarms and event requests that require acknowledgment. Click Enterprise Schedule Queue to open the Enterprise Schedules Queue. This shows the Enterprise Schedule updates queued for sending.

Workspace toolbar —The workspace toolbar contains buttons related to the page that is currently displayed in the main workspace.

The RC-WebView interface provides a breadcrumb navigation trail that appears below the menu bar. Click the links in the breadcrumb trail to easily navigate forwards and backwards in the RC-WebView interface.

Figure: Breadcrumb navigation links

Menu bar — The menu bar is located at the top right of the banner, and is accessible throughout the application.
