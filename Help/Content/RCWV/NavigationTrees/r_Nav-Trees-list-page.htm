Navigation Tree List page

Use the Navigation Trees List page to add, edit, copy, rename, or delete Navigation Trees.

Click the Navigation Trees node in the System Tree to access the Navigation Trees List page.

Use the following options to configure a Navigation Tree:

Add to create a Navigation Tree using the name you enter in the Navigation Tree Name input box.

The following options apply to the Navigation Tree that you select in the Navigation Trees list.

Edit to open the Navigation Tree Settings page where you can modify the connected systems, devices, and objects.
Copy to copy an existing tree and modify it to meet your requirements.
Rename to change the name of the Navigation Tree.
Delete to delete the selected Navigation Tree. Hold the Shift key to select and delete multiple trees.
