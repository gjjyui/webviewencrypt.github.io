Workspace toolbar buttons

The workspace toolbar contains buttons that are related to the page that is currently displayed in the main workspace. The content of the toolbar is determined by the user's permissions.

Toolbar buttons and their functionality are listed below.

Sets the System Group or Navigation Group scaling mode to accommodate monitor sizes and user preferences. Click the scaling icon to change the view to:

  • Fullsize—Maintains the native size of the Group graphic.
  • Scaled—Sets the Group graphic size as large as the current browser window will allow while still maintaining the aspect ratio.
  • Stretched—Sets the Group graphic size as large as the current browser window will allow while ignoring the aspect ratio.

Click to filter the content displayed in the Audit Trail or Alarm History worksheets.

Click to edit the contents of an Enterprise Schedule or Trend Log. The user must have modification permissions in order to edit an Enterprise Schedule or Trend Log.

Deletes the selected item.

Click to show or hide the columns that appear in a worksheet.

Cancel or undo any changes without saving.

Click to save changes. The floppy disk icon appears in red when there are changes to be saved.

Preview a Navigation Group.

Zoom in

Zoom out

View an Enterprise Schedule or Trend Log.

Select all check boxes.

Clear all check boxes.

Displays an After Hours Report.

Approves an After Hours request.

Rejects an After Hours request.

Change a Trend Log to Update Mode.

Load new data.

Acknowledge alarms.

Send any changes to the controller. The button is highlighted in red when there are unsaved changes.

Export the worksheet data as a .csv file. It is downloaded to your browser downloads folder.
Export the worksheet data as an Excel file. It is downloaded to your browser downloads folder.

Adds new items to a worksheet.

Sends the selected program to multiple selected controllers in the Programs worksheet.

Allows you to select Open or Save files:

  • Open — Opens a dialog box where you can select a Control-BASIC program to open in the editor.
  • Save —Opens a dialog box where you can select options to save the Control-BASIC program.

Toggles between Watch and Trace in the program editor:

  • Watch — Expands the left pane of the editor to display the current values of any local program variables along with any database objects used in the program that were added to the Watch List.
  • Trace— Reduces the left pane of the editor to display the program execution trace. Program lines with a corresponding T have been executed since the trace bits were cleared. Program lines with no corresponding T were not executed because the program gave instructions to bypass these lines.

Toggles the program editor to display the object mnemonics or the object names.

Toggles the program editor to append the data point's present value or the time function's present value in angled brackets after each point. Present values are continually updated when displayed. Display values are updated every five seconds.

Toggles the program editor to display a dynamic listing of possible Control-BASIC functions when entering text into a program.

Displays a find and replace dialog box to replace single instances or all instances of the entered text within a program. All instances of the entered text are highlighted in the editor.

Renumbers the line numbers of the currently displayed program in increments of 10. Any code references to specific line numbers are also updated.

Opens the Options dialog box for the Control-BASIC editor.

Back up selected devices. If a single third-party device is selected, you will be prompted for the device's password if required.
Restores selected devices. If a single third party device is selected, a password prompt for the device's master password will be given if needed.
Opens the Auto Network Backup dialog box.
Displays the results of the last backup or restore operation.
