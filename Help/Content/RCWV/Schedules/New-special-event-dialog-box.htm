New Special Event dialog box

The New Special Event dialog box opens when you click a date in the Special Events pane on the right of the Schedule and then click Create in the pop-up menu. The New Special Event dialog box contains the following settings, organized by group:

Event Detail

Name

The name of the currently selected Special Event.

Rank

The rank of the current Special Event. Modify it by using the drop-down arrow to select from a list. The rank determines which Special Events takes precedence if more than one falls on the same date. All Special Events take precedence over the weekly Schedule.

Ranks are predefined as:

  • 1 — Highest
  • 16 — Lowest
  • 17 — Weekly Schedule
  • 18 — Unscheduled

Note: Ranks 1–16 are available for programmers to assign to Special Events.

Period

Date/Range

Choose a specific date for the Special Event or for a date range click the Calendar icon to the right of the Start and End boxes to select the beginning and end range.

  • D, M, or Y to repeat the Special Event on every Day, Month, or Year.
  • * to always repeat this Event.
  • Advanced Options to choose a specific Day of Month, Month, or Day of Year.
  • Auto Delete so the Special Event is automatically deleted from the controller upon expiry.

Recurrence

Use the drop-down arrows to select from a list:

  • Month—Month, odd months, even months, or any month.
  • Week of Month—Days 29 to 31, Last 7 days, or Any Week.
  • Day of Week—Monday through Sunday, or Any day.

Calendar

Use the drop-down arrow to see a list of all BACnet Calendars that are available in the controller. Only one BACnet Calendar can be linked to a single Special Event.

Override

Control the status of a Schedule with any BACnet object on the network. Typically, this feature is used to link multiple Schedules to a remote Schedule (acting effectively as a Master Schedule), but any point may be used in the Override box. This is a proprietary extension to the BACnet standard.

Time Values

Displays the Special Event and any Start and Stop times that have been entered in the Day or Time and Value columns. Use the drop-down arrow to see the list of available options, or enter a value. Up to seven start/stop pairs can be created for a total of 15 Events. For binary BACnet Schedules use the following:

All On

The Schedule value is set to On for the entire day, with a Relinquish at the end of the day.

All Off

The Schedule value is set to Off for the entire day, with a Relinquish at the end of the day.

Start/Stop

Drag from the green column to enter into the Day field.

Relinquish

Default for unselected areas.
