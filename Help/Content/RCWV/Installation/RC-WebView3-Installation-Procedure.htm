Run the RC-WebView installer

Use the RC-WebView InstallShield Wizard to guide you through installing RC-WebView.

Before installing RC-WebView, ensure that the installation prerequisites are satisfied.

Use the following steps to install RC-WebView using the wizard:

  1. Double-click the RC-WebView install file. The InstallShield Wizard displays the supporting files to be installed. Click Install.
  2. A message may appear to request that the system be restarted. Click Yes to restart or No if restarting later.
  3. If a restart was required then the InstallShield Wizard appears and lists the pending items to be installed.
  4. Click Install to begin the RC-WebView Database Setup.
  5. In the Configuration Database dialog box, click Yes to install the Microsoft SQL Server database or No if you already have an existing SQL Server or a custom SQL installation. Clicking No skips the SQL installation.
  6. The SQL Server 2012 Setup dialog box displays. Click Next and the InstallShield Wizard appears and begins the RC-WebView Setup installation.
  7. Click Next on the InstallShield Wizard to begin the installation.
  8. Read the License Agreement and select "I accept the terms in the license agreement" and click Next.
  9. Specify the Destination Folder for the website files and click Change to browse to another location on the computer or click Next to accept the default location and to continue.
  10. Specify the Web Application Configuration and hosting information settings by selecting one of the following:
    • Default — Use for most RC-WebView configurations. Use Port: 8180, any IP address (*) and does not require a Host name. Note: Do not use the default if the target server with Internet Information Services (IIS) is already using Port: 8180, in which case an alternate port must be specified in Custom Settings.
    • Custom — Use for specific Port numbers, IP addresses and Host names. Ensure that the network and server firewall are configured to the custom specifications in order to facilitate communication.

  11. Specify the database server and authentication method:
    • Database Server—The name and credentials are automatically entered if Yes was previously selected to install a (local) Microsoft SQL Server database. If using a custom or remote SQL Server installation, then use the dropdown arrow to select from a list or click Browse to locate the server.
    • Connect using—This only appears if you are connecting to a customized instance of Microsoft SQL Server that does not allow SQL-based authentication. Select the authentication method used to make the connection: Windows authentication to connect with Windows ID and password, or Server authentication to connect to Microsoft SQL Server using SQL-based authentication.
  12. Type the Login ID and Password for the SQL administrator account. Leave as the default value if using the Microsoft SQL Server database created during the RC-WebView installation process.
    • Name of database catalog—The name of the RC-WebView database that resides within the SQL Server. If you just installed the Microsoft SQL Server database as part of the RC-WebView installation, the name defaults to RC-WebView. If using a custom or remote SQL Server installation, click Browse to locate and display the database name. Click Next.

  13. Type and confirm the Password for the Administrator account. Click Next to see the InstallShield Wizard.
  14. Click Install to start the installation, click Back to review any previous setting, or click Cancel to exit the Wizard and stop the installation procedure.
  15. A message appears and requests the entry of the license file. Use Browse to locate the file (i.e., RCWebView.LF) that was purchased and stored on your computer. If you do not have a license, contact a Reliable Controls Authorized Dealer for a license.
  16. If this is an upgrade to an existing installation of RC-WebView then choose Yes and Yes again to import the previous configuration settings, otherwise choose No.
  17. When the installation has completed the InstallShield Wizard provides the option of creating a Windows Installer Log. Select the check box if a log is required and or click Finish to exit the InstallShield Wizard.
