Installation prerequisites

When considering a new installation of RC-WebView, it is important to review the anticipated level of usage. In general, as the number of simultaneous users and hosted Enterprise Websites increases, the investment in operating system, server hardware, and network infrastructure must also increase.

The following factors impact the speed and quality of the operator interface.

  • The anticipated level of usage.
  • The number of simultaneous users and Enterprise Websites.
  • The choice of operating system.

Recommended Server Requirements

  • All hardware rated for continuous server operation.
  • Intel Xeon E3 3.10 GHz or better.
  • Four cores must be available for RC‑WebView. For multiple products installed on a single computer, you need two cores per product plus at least two cores for the SQL Server database.
  • x64 system architecture.
  • 16 GB RAM.
  • Hardware RAID in a redundant performance configuration with 10K RPM SAS drives or better.
  • 15 GB free hard disk space.
  • Internet connection for software download and installation.
  • Server-quality gigabit network interface card.
  • Windows Server 2012 R2 Standard–Windows Server 2019.
  • Microsoft SQL Server 2014 (x64)–Microsoft SQL Server 2019.
  • Router with firewall enabled. Broadband network connection with minimum upload speed of 1 Mbps.
  • Maximum configuration: 500 connections per server.

Browser Support

  • Google Chrome.
  • Microsoft Edge 83 or later.
  • Safari.
