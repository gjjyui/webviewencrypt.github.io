Login Info page

Use the Login Info page to specify a user's:

  • Log in credentials.
  • Status (Enabled or Disabled).
  • Assigned Navigation Tree.
  • Assigned After Hours Template.
  • Home Group when logging on using a browser or the myControl app.
  • Access to controller systems.
  • Access to Schedules.
  • Roles

To display a user's Login Info page, expand Users in the Navigation Tree and then click the user's name.

Note: An Inherited Role icon appearing beside any of the user's parameters indicates that the user inherits the setting from an assigned user role. Hover over the icon to see which role provides the setting.

The following table describes the options on the Login Info page for a user:

Username

Name of the user that appears in the System Tree under Users.

User Type

Specifies the authentication method for the user: local authentication or Microsoft Active Directory.

Enabled/Disabled

The user's status in RC-WebView. Disabled status restricts the user from logging into RC-WebView.

Name

The name of the user displayed in the Navigation Toolbar.

Email Address

The user's email address.

New Password

The user's password.

Verify Password

Confirmation of the user's password.

Password reset required

The user must change their password the next time they log on.

Comments

Additional comments about the user.

Navigation Tree

The default Navigation Tree that displays when the user logs on.

After Hours Template

Determines the report format used to display After Hours information for that user.

Home Group

Home Group settings determine what displays when a user first logs on. In the Home Group area, select either Navigation Group or System Group.

  • Navigation Group —When selected, choose from available Navigation Groups in the Navigation Group drop-down box.
  • System Group — When selected, choose a system from the list of systems the user has access to using the System drop-down box. In the System Group box, select a System Group available on the selected system.

myControl Home Group

myControl Home Group settings determine the System Group that displays when a user logs using the myControl app.

  • System — Select a system from the drop-down list.
  • System Group — Select a System Group from the drop-down list.

Permitted Systems

Determines the systems the user can work with to view and manage data. Click to choose between All Systems or one or more individual systems that have been defined in RC-WebView.

Permitted Schedules

Determines the Schedules that the user can access and manage. The user will only have access to the selected Schedules within that Enterprise Schedule.

Click the links to display the Enterprise Schedule Permissions page where you can select one or more Schedules in a hierarchy view. The user will only have access to the selected Schedules. This setting does not affect the user's ability to work with Schedules outside of Enterprise Schedules.

Roles

Displays the user roles applied to this user. Click the X to remove a role from this user, or click the + to add a role to this user.
