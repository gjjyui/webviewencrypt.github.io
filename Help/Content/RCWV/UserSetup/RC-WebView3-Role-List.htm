Role List page

View and manage user roles in the Role List page. Create user roles to apply logon information and permissions to multiple users at once.

Access the Role List page in one of two ways:

  • Click Users in the System Tree to open the User List, then click Role List.
  • Click Role List from the Login Info or Permissions page for any user.

Buttons

There are two buttons on the Role List page:

Role List

Click this to return to the Role List if you navigate away from the Role List.

Add Role

Click to open the Add Role wizard.

Icons

The following toolbar buttons are on the User List page:

Select/clear all check boxes.
Modify Enterprise Schedule permissions. This is only accessible if at least one role is selected.
Delete the selected roles. This is only accessible if at least one role is selected.

Drop-down menus

Use the following drop-down menus to assign various features to users. Most of these drop-downs are only accessible once a user is selected.

Assign Navigation Tree

Assign a previously-created Navigation Tree to the selected users.

After Hours Template

Assign a previously-created After Hours Template to the selected users.

Set Home Group

Assign a previously-created Navigation Group to be the home group for the selected users.

Set myControl Home Group

Assign a previously-created System Group to be a myControl Home Group for the selected users.

Restrict to System(s)

Restrict the users to selected systems.

Search

Search for a user based on settings applied for that user. Enter a search term and click the dropdown button to apply a filter to the search.

Filter by System(s)

Select a system to view users that have access to that system.

Worksheet columns

Role Name

The name of the user that appears in the System Tree under Users. Click to access the Login Info page for the user.

Permitted Systems

The systems that this role is permitted to access.

Navigation Tree

The Navigation Tree assigned to this role.

After Hours Template

The After Hours Template assigned to the role.

Home Group

The Home Group is what displays when a user with this role first logs on. It can be a System Group or Navigation Group.

myControl Home Group

The System Group that displays when a user logs using the myControl app.

Enabled

Enable or disable the role by selecting or clearing the check box. If a role is disabled, it can be applied to users without actually applying any settings.
