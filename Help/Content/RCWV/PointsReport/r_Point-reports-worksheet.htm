Point Reports worksheet

Use the Point Reports worksheet to add, modify, or delete individual Point Reports, or to open a Point Report listed in the worksheet.

To access a Point Reports worksheet, click a system node in the System Tree to open the menu, and then click Point Reports.

The Point Reports worksheet includes the following toolbar items:

Select All — Selects all of the check boxes for the individual Point Reports listed in the Point Reports worksheet. This icon acts as a toggle with Deselect All.
Deselect All — Clears all of the check boxes for the individual Point Reports. This icon acts as a toggle with Select All.
Save — Saves any changes you make to the entries in the worksheet.
Unsaved Changes - Saves any changes you made after your last save. This icon appears when you change an entry in the worksheet.
Cancel — Reverts any unsaved changes.

Export — Exports the contents of the worksheet to a .csv file.

This option is not available when the enforce change approval option is enabled on the Enterprise Website Security page.

Export as Excel — Exports the contents of the worksheet to an Excel file.

An exported Excel file is digitally signed using the signature certificate specified on the Enterprise Website Security page.
Toggle Columns — Opens the menu to choose the columns displayed in the Point Reports worksheet. Select or clear check boxes to either display or hide columns.
Add — Opens the Create Points Report dialog box. Add a name for the new Points Report and click OK to add a new report instance to the worksheet.
Delete — Deletes the selected Point Reports from the worksheet.

Worksheet columns

The Point Reports worksheet contains the following columns:

Instance

A number identifying the individual Point Report instance. Click the number to open the Point Report.

Name

The name of the individual Point Reports created in RC‑WebView and RC‑Studio.

Description

The optional description of the Point Reports.

Last Modified

Indicates the day and date of the latest modification to the entry in the worksheet.

Point Count

Indicates the number of points included in each of the individual Point Reports.

Security

The security level for the Point Report. Users with an Access Level equal to or greater than the security level can open the Point Report.
