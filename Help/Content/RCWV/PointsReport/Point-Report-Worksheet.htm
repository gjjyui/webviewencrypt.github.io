Individual Point Report worksheet

Use the Point Report worksheet to view and manage a customized, filtered list of system objects. A Point Report allows you to make batch changes to multiple objects at one time.

To open an individual Point Report worksheet:

  • Click a Point Report reference in a System Group or Workstation group.
  • Click a Point Report entry in a row in the Point Reports worksheet, which is available by clicking a system node in the System Tree and then clicking Point Reports in the flyout menu.
  • Click a Point Report entry in the Navigation Tree or Navigation Group.

An individual Point Report worksheet includes the following toolbar items:

Select All — Selects all of the individual Point Reports listed in the Point Reports worksheet. This icon acts as a toggle with Deselect All.
Deselect All — Clears all of the check boxes for the individual Point Reports. This icon acts as a toggle with Select All.
Save — Saves any changes you make to the entries in the worksheet.
Unsaved Changes - Saves any changes you made after your last save. This icon appears when you change an entry in the worksheet or if you change which columns are displayed in the worksheet.
Cancel — Reverts any unsaved changes.
Edit Values — Opens the Edit Selected Points dialog box which allows you to perform bulk edits of selected points values and to change their state to Auto or Manual. (See the note below.)

Export — Exports the contents of the worksheet to a .csv file.

This option is not available when the enforce change approval option is enabled on the Enterprise Website Security page.

Export as Excel — Exports the contents of the worksheet to an Excel file.

An exported Excel file is digitally signed using the signature certificate specified on the Enterprise Website Security page.
Toggle Columns — Places the worksheet header in edit mode with check boxes beside each column name. Select the check boxes for the headings that you want to display in the worksheet, and clear the check boxes for the columns that you want to hide. Click Save to implement the changes. You can add or remove columns at any time by clicking the Toggle Columns icon. (See the note below.)
Delete — Deletes the selected Point Reports from the worksheet. (See the note below.)
Note: The toolbar item is only enabled and available if Point Reports is selected in your Modify permissions.

Worksheet columns

The following columns are available to display in an individual Point Report worksheet:

Device Name

The name of the device where the related object is defined.

Name

Name of the object.

Display Name

A manually entered name to associate with the corresponding point specific to the Point Report.

Description

A manually entered description for the point.

Mnemonic

Protocol mnemonic for the object.

Value

Present value of the object. Click in the cell to edit the value.

Unit

The engineering unit associated with the point.

Auto/Manual

Control state of the object. Click in the cell to edit the state.

Timed Override

If Auto/Manual is set to Timed, then the value displayed is the amount of time remaining before returning to automatic control. Click the cell and type a new value in order to adjust the time remaining.

Priority

Priority of the present value.

Changed

Displays a checkmark indicating unsaved changes have been made to the corresponding object.

Error Status

Displays an error for the corresponding point if one exists.

Note: If a Point Report's configuration is changed via RC‑Studio 3.6.3 or earlier, any RC‑WebView-specific features will be overridden. That is, Display Name values will be removed and any worksheet columns not available in RC‑Studio will become visible in RC‑WebView even if they had previously been hidden.
