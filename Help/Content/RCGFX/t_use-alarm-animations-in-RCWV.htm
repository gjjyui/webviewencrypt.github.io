Use alarm animations in RC‑WebView

You can add alarm animations to Navigation Groups in RC‑WebView using the Insert Animation pane. After adding the animation, you need to ensure that the correct base URL is specified.

Complete the following steps to add an alarm animation to a Navigation Group and to specify the required base URL on the Enterprise Website Settings page. The following instructions assume that you have added a Navigation Group.

  1. Click Add in the Navigation Group toolbar and click Animation in the drop-down menu to open the Insert Animation pane.

  2. To add an alarm animation, select Upload RCA From Workstation and browse to the location of the RCA file.

    By default, the files in the folder location are filtered to show only RCA files. The uploading of JSON files is not supported.

    If an alarm animation has been previously uploaded, select Select Previously Uploaded RCA and click the down arrow to display the list of animations to choose from.

  3. Click an RCA file name, and click Open to add the file path to the input box and to display the animation in the preview area in the Insert Animation pane.

  4. Drag the animation from the preview area to the Navigation Group canvas.

    When you drag and drop an animation on the Navigation Group, it retains its default size.

  5. Ensure the Base URL setting on the Enterprise Website Settings page matches the base URL a user sees in their browser when they view the Navigation Group.

    For example, if the user accesses the Enterprise Website at http://rcwebviewdomain:8180/sitename, you need to ensure that the Base URL setting is http://rcwebviewdomain:8180/.

For information about setting the options for a particular alarm animation, click Edit to open the Edit Animation pane and view the information on the Help tab.

You may notice the RC-WebView Enterprise Website URL input box on the Attributes tab when you edit the settings for the animation. An entry is not required when you add alarm animations to Navigation Groups in RC‑WebView. In addition, this setting is not required if you add alarm animations to a System Group or Workstation Group that will be viewed in RC‑WebView. For example, you can configure an alarm animation in RC‑Studio without seeing the alarm data until you view it in RC‑WebView. This setting is required when you want to view alarm data in the animations in System Groups or Workstation Groups in RC‑Studio.
