INTERVAL

Facilitates repetitive execution at a specified interval of time.

INTERVAL, TIME-ON, and TIME-OFF all return a 0 and timers begin with 0 on the first scan of a program. Be mindful of equipment that is cycled or staged based on timers when you send programs to controllers that actively control equipment.

Function

Causes repetitive execution at a specified interval of time.

Format

INTERVAL( exp )

Where

exp is the time of the interval, expressed as 00:00:00 or as a positive real number. See TIME.

Example

10 IF INTERVAL( 0:10:00 ) THEN GOSUB 100

20 REM AV1 in next line will increment
   by 2 each hour

30 AV1 = AV1 + 2 * INTERVAL( 100 )

40 END

100 REM This subroutine is executed
    every 10 minutes

190 RETURN

In Line 10 the function INTERVAL( 0:10:00 ) has a value of 1 for one scan every 10 minutes.

In line 30 the function INTERVAL( 100 ) has a value of 1 for one scan every 1 hour.
