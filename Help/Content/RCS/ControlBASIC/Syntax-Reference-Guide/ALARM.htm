ALARM

Generates an alarm based on an analog condition.

Not available in the SMART-Space Controller.

Statement

Generates an alarm based on an analog condition.

Format

ALARM exp1 > exp2 , diff , string

ALARM exp1 < exp2 , diff , string

Where

 

 

exp1 > exp2 and exp1 < exp2 are the alarm conditions that, when true, generate alarms. The type and priority of the alarm are determined by the last executed ALARM-TYPE statement.

diff is the alarm switching differential in units of exp1 and exp2 between the alarm state and the normal state.

string is the alarm message, with a 69-character maximum.

Example

10 ALARM BLR-HWS-T < 60 , 5 ,

   Boiler HWS Temp Low

 

The above line of code generates an alarm message that reads, "Boiler HWS Temp Low" when BLR-HWS-T falls below 60. The alarm state changes to Reset when BLR-HWS-T rises above 65.

Trigger and automate an action

Control-BASIC can trigger an action in conjunction with the alarm message. You can trigger more than one action per statement, but they must be separated by a space to execute. If an action string contains spaces, such as a file path, enclose the string in double quotes for Control-BASIC to recognize it as a single entity.

To activate an action on a workstation, set the Alarm Enable column on the Alarm Configuration worksheet to Worksheet and Pop-up, and set the Action Enabled column to Unconditional for automatic execution, or set it to Prompted if the operator must indicate approval before the action proceeds.

There are two types of possible actions:

  • #G — Launches the point or System Group.

#G (point)

Launches the point or System Group named in the parentheses.

Example 1

 

10 DALARM AH3-FRZ , 60 , AH-3
Freeze Stat tripped #G(3000BI1)

Displays the worksheet for Binary Input 1 on Panel 3000.

Example 2

 

10 ALARM AH-3-SAT > 50 , 2 , AH-3 S/A      

   Temp High ALarm #G(7000GRP2)

Displays System Group 2 on Panel 7000.
  • #R — Launches the executable.

#R (file/path)

Launches the executable named in the parentheses. If the executable does not reside in the system’s root directory, provide a full path, or add the executable to the Microsoft Windows Path statement to ensure it is found.

Example

10 DALARM AH3-FLT , 60 ,
   AH-3 Filter is dirty

#R(“C:\RC-Studio\Headquarters\Filter-Repl-Instr.doc”)
 

When the alarm condition occurs, the program on the PC associated with .doc files, such as Microsoft Word, opens the named document.
